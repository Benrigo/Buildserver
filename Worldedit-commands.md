# Nice Worldedit commands

### Wege commands

###### Rietweg/erdigerweg
<details>
  
```
//brush set sphere 2 40%coarse_dirt,30%packed_mud,20%mud_bricks,10%rooted_dirt
```

```
/mask grass_block
```

### Felder machen
<details>
  
```
//brush set sphere 2 farmland
```
Mit der Mask
```
/mask grass_block
```

```
//brush set sphere 59:7
```
Mit der Mask
```
/mask >60
```
</details>
